## 💻 로컬 실행 방법

```bash
# 1. 레포 클론
git clone https://github.com/sosiluv/sosiluv.github.io.git
cd sosiluv.github.io

# 2. Ruby, Bundler 설치 (생략 가능)

# 3. 의존성 설치
bundle install

# 4. 로컬 서버 실행
bundle exec jekyll serve
```

사이트는 [http://127.0.0.1:4000](http://127.0.0.1:4000)에서 확인할 수 있습니다.
